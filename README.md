# AI虚拟人实时交互项目工程
这个是一个可落地的AI虚拟数字人实时交互技术项目，虚拟人基本无延迟，对接大模型有：GPT、百度文心一言、离线大模型、星火大模型等等，虚拟人可以进行24小时直播或实时语音交互，能成为你的AI虚拟员工。

## 使用方法
1.下载工程文档，安装C++环境
2.配置本地知识库序列号和相关大模型API
3.运行exe即可
